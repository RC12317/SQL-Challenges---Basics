# Purpose
•	Practice and reinforce MySQL fundamentals

•	Improve problem-solving skills using SQL

•	Prepare for interviews, assessments or real-world database tasks

•	Serve as a personal reference for common SQL patterns

•	SELECT statements

•	Basic data filtering and sorting (WHERE, ORDER BY, and LIMIT)

•	Aggregate functions (COUNT, SUM, AVG, MIN, MAX)

•	GROUP BY and HAVING

•	JOINs (INNER, LEFT, RIGHT)

•	Conditional logic with CASE expressions

•	Basic subqueries 

•	String and date functions
